如何何纯代码为WordPress文章页添加面包屑导航

发表于2019-06-20|更新于2019-09-11|杂谈
|阅读量:

如何纯代码为 WordPress 文章页添加面包屑导航，我们主要通过修改主题 functions.php、single.php、main.css 三个文件来实现。

以下方法转自@蝈蝈要安静。注意以下并非本站所用的面包屑，无法于本站一样，但基本原理相同，经供参考

一、修改 functions.php

首先在自己的主题文件中找到 functions.php，并在文件中添加下面代码。

//面包屑导航生成函数
function qgg_breadcrumbs(){
if( !is_single() ) return false; $categorys = get_the_category(); $category = $categorys[0];
return ‘当前位置：‘.get_bloginfo(‘name’).’ > ‘.get_category_parents($category->term_id, true, ‘ > ‘).get_the_title();
}
