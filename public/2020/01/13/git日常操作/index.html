目前在我工作中用到的版本控制工具有 SVN 和 Git,代码开发主要以 Git 为主，Git 的命令有很多不过只是常规操作也就那么 6 个，但是要熟练使用那就需要记住很多命令了


Git 操作流中的专业术语，简单解释一下

  • Workspace：工作区
  • Index / Stage：暂存区 （add 操作后放置的地方）
  • Repository：仓库区（或本地仓库执行 commit 操作后保存到的地方）
  • Remote：远程仓库（线上仓库）
